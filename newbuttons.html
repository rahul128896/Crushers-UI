Animated Border Button

Border animates around the button on hover

View Code 
<button class="btn btn-border-anim"><span>Animated Border</span></button>
.btn-border-anim {
  background: none;
  border: 2px solid transparent;
  color: var(--purple-color);
  position: relative;
  overflow: hidden;
  z-index: 1;
}
.btn-border-anim span {
  position: relative;
  z-index: 2;
}
.btn-border-anim::before {
  content: '';
  position: absolute;
  top: 0; left: 0;
  width: 100%; height: 100%;
  border: 2px solid var(--purple-color);
  border-radius: 9999px;
  transform: scale(1.1);
  opacity: 0;
  transition: all 0.4s cubic-bezier(0.23, 1, 0.32, 1);
  z-index: -1;
}
.btn-border-anim:hover::before {
  transform: scale(1);
  opacity: 1;
}
.btn-border-anim:hover {
  color: var(--purple-dark);
}

Skewed Button

Button with a dynamic skew effect

View Code 
<button class="btn btn-skew"><span>Skewed</span></button>
.btn-skew {
  background: linear-gradient(45deg, var(--purple-color), #673ab7);
  color: white;
  transform: skewX(-15deg);
  border: none;
}
.btn-skew span {
  display: inline-block; /* Needed for inner transform */

  transform: skewX(-15deg) scale(1.05); /* Add scale on hover */
}
